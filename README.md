# ege-tire-kutsan-web

Developed as part of a 1st-year Computer Programming academic project, this is an interactive guide website built with modern web technologies that comprehensively introduces Ege University, Tire Kutsan Vocational School, and the historic Tire district of Izmir.

## Technologies Used
The following technologies and tools were used during the project development process:
- **HTML5:** Structural content and semantic markup.
- **CSS3:** Modern, responsive interface design.
- **JavaScript:** Menu interactions and dynamic user experience.
- **Font Awesome:** Icon library.
- **Google Maps API:** Interactive map embeds for points of interest.

## Project Structure and Modules
The system consists of three main sections and a central landing page that connects them:

1. **Main Landing Page (`index.html`):** The central hub that welcomes the user and directs them to the three main sections (Ege Uni, MYO, Tire).
2. **Ege University Section (`1-ege-uni`):** General overview, campus facilities, and academic vision of the university.
3. **Tire Kutsan MYO Section (`2-tire-kutsan`):** Detailed information about the school's history, departments, and student life.
4. **About Tire Section (`3-tire-hakkinda`):** A comprehensive guide covering the district's history, places to visit, traditional handicrafts, and unique local flavors.

# ege-tire-kutsan-web

Bu proje Bilgisayar Programcılığı 1. sınıf akademik projesi kapsamında, Ege Üniversitesi, Tire Kutsan Meslek Yüksekokulu ve İzmir'in tarihi Tire ilçesini kapsamlı bir şekilde tanıtan, modern web teknolojileriyle geliştirilmiş interaktif bir rehber sitesidir.

## Kullanılan Teknolojiler
Proje geliştirme sürecinde aşağıdaki teknolojiler ve araçlar kullanılmıştır:
- **HTML5:** Yapısal içerik ve semantik etiketleme.
- **CSS3:** Modern, duyarlı (responsive) arayüz tasarımı.
- **JavaScript:** Menü etkileşimleri ve dinamik kullanıcı deneyimi.
- **Font Awesome:** İkon kütüphanesi.
- **Google Maps API:** Önemli noktalar için interaktif harita yerleştirmeleri.

## Proje Yapısı ve Modüller
Sistem, üç ana bölümden ve bunları birbirine bağlayan merkezi bir giriş sayfasından oluşmaktadır:

1. **Ana Giriş Sayfası (`index.html`):** Kullanıcıyı karşılayan ve üç ana bölüme (Ege Uni, MYO, Tire) yönlendiren merkezi köprü.
2. **Ege Üniversitesi Bölümü (`1-ege-uni`):** Üniversitenin genel tanıtımı, kampüs olanakları ve akademik vizyonu.
3. **Tire Kutsan MYO Bölümü (`2-tire-kutsan`):** Okulun tarihçesi, bölümleri ve öğrenci yaşamı hakkında detaylı bilgiler.
4. **Tire Hakkında Bölümü (`3-tire-hakkinda`):** İlçenin tarihi, gezilecek yerleri, el sanatları ve eşsiz lezzetlerini içeren kapsamlı rehber.
